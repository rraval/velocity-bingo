VeloCity Bingo
==============

Drinking game to accompany pitch nights at VeloCity. See the sample PDF files
for a quick demo.

Usage (Generates 20 pages, 4 bingos per page):

```
$ python3 bingo.py > bingo.tex
$ lualatex bingo.tex
```
